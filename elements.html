A Beginner's Guide on Web Development

Basic HTML Tags

<html>

This element defines an html document

<head>

This element defines an html document

<head>

This element contains metadata/information for the document

<title>

This element defines a title for your html document

<body>

This element defines an html document's body

<h1> to <h6>

This element defines HTML headings

<p>

This element defines a paragraph

<br>

This element inserts a single line break

<a href>

This element specifies the destination of a hyperlink.

<img>

This element embeds an image within your HTML document.

Lists and Tables

<table>

This element defines a table. Below is an example:

Age Name School
17 Malcolm Yay
20 Cyp Wooohoo
26 Jungkook Oppar

<th>, <tr>, <td>

These elements respectively define the header cell, a row, and cell in a table.

<ul>, <ol>, <li>

These elements respectively define an unordered list, an ordered list and a list item.

Here is an example of an unordered list:

  • Apples
  • Bananas
  • Woohoo

Here is an example of an ordered list:

  1. Apple
  2. Banana
  3. Strawberry

Styles and Semantics

<style>

This element defines style information for an HTML document

<div>

This element defines a general section in a document

<span>

This element marks up a part of a text or part of the document

<header>

This element defines a header for a document or section

<footer>

This element defines a footer for a document or section

<flex>

This element refers to CSS flexbox, used for arranging items within a container.
