What is CSS?

CSS is an acronym for Cascading Style Sheets, used for designing and altering the visual appearance web applications and websites. It is a style sheet language that controls the presentation and formatting of documents.


There are three types of CSS:

External CSS: Styles are written in a separate .css file and linked to HTML documents. This enables consistent styling across an entire website.


Internal CSS: Styles are embedded within the <head> section of an HTML document.


Inline CSS: Styles are directly applied to individual HTML elements using the <style> attribute.
